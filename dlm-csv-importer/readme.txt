=== Download Monitor CSV Importer ===

Mass import up to thousands of Downloads into Download Monitor with the CSV Importer.

== Installation ==

To install this plugin, please refer to the guide here: [http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation)

= Usage =

After installing you can import your CSV file via the WordPress Tools > Import screen.

See https://www.download-monitor.com/documentation/ for documentation.
